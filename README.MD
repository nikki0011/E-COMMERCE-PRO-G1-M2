Este proyecto es una aplicación web de e-commerce desarrollada con React y TypeScript que permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre un listado de Productos. 

## 🦖 Demo

mira la demo del proyecto [aqui](link)

## 🚀 Librerías utilizadas

- React
- TypeScript
- Vite
- React Router
- React-Hook Form
- Flowbite-React
- Tailwind
- react-icons
- sweetalert 2

## 📦 Pasos para usar el proyecto

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/nikki0011/E-COMMERCE-PRO-G1-M2.git
   ```
2. **Instalar dependencias:**
   ```bash
   pnpm install
   ```

3. **Iniciar el proyecto:**
   ```bash
   pnpm run dev
   ```
